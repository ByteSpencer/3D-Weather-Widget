3D Weather Widget Design

<img src="https://github.com/user-attachments/assets/77e99e6b-b56f-46ad-8570-b8bac1c863e6" width="300">


**What?**
* An animated responsive weather dashboard built with HTML, Tailwind CSS, &Three.js

* Automatically fetches live weather conditions using user's coordinates via the browser (or accepts manual search input) ,  has unit switching (°C/°F), + transparent background for widget embedding

**How to embed into Notion, Googlesites, + ?**

1. copy this link: https://bytespencer.github.io/3D-Weather-Widget/
2. go to location, click embed (notion = /embed), & then paste it into where it says "link"
3. resize it how you'd like & done!
